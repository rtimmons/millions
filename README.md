Getting Started
---------------

    brew install rbenv ruby-build
    # follow rbenv install instructions re shell profile additions
    
    rbenv install 2.1.0
    # restart shell
    
    git clone git://github.com/carsomyr/rbenv-bundler.git ~/.rbenv/plugins/bundler
    # restart shell
    
    gem install bundler
    rbenv rehash
